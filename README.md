# **CyberneticFuture(tmp)**  
<br>  
<br>  

## **ゲーム設計**  
<br>  
<br>  
<br>  



### **コンセプト**  
近未来のサイバーパンクな世界線で三人称視点でキャラクターを操作し敵陣にある`タンク`に`エナジー`を注入して注入した`エナジー`量を競う完全3DのオンラインMOBA  
<br>  
<br>


### **世界観**  
近未来の地球で流行ってるサバイバルゲームという設定
<br>  
<br>  


### **キャラクター**  
- 三人称視点でキャラクターを操作[^1]  
- 使う装備セットによってステータス(ライフ,移動速度,`エナジー`量,武器など)が異なる[^1]  
- 装備セットにはそれぞれ特有の近未来的な武器(ライトセーバー,レーザーガン,エナジーブースト,高エネルギーグローブなど)がある[^2]  
- 各装備セットにはそれぞれタイプ(遠距離攻撃型,妨害型,支援型,近距離攻撃型,特殊型)の武器が存在する[^2]  
- 武器はそれぞれのキャラクターが持つ`エナジー`を消費して戦う[^2]  
- 各装備には一定時間に一度使える必殺技がある[^3]  
- キャラクターは3つまで`ガジェット`を装備できる[^4]  
- 4vs4で協力しあいながら戦う[^1]  
- キャラクターのアバターはカスタマイズできる[^1]  
<br>  
<br>  


### **フィールド**  
- 近未来サイバーパンクなフィールドでプレイする[^2]  
- フィールドには様々な種類がある(中国風サイバーパンク,NY風サイバーパンク,ドバイ風サイバーパンク,ヨーロッパ風サイバーパンク,日本風サイバーパンクなど)  
- フィールドは毎回ランダムで決まる[^2]  
- 各フィールドによって出現する`メカ`や`ロボ`が異なる[^3]  
- フィールドには定期的に`ロボット`が出現し倒すと各`ロボ`に応じた`エナジー`を取得できる[^3]  
- フィールドのそれぞれの端から各チームの4人が配置されゲームスタート[^1]  
- フィールドの特定の場所に行くとスタート地点に戻れる[^3]  
- フィールドにある建物の中には入れない[^1]  
- フィールドの広さは東京ドームぐらいの規模[^1]  
- フィールドは円形[^1]  
- フィールドには縁がありそれより奥には行けない[^1]  
<br>  
<br>  


### **特有の用語**  
- `エナジー` => 武器を使用するのに必要なもので武器を使用すると消費される一方敵陣にある`タンク`に注入してゲームの得点になる  
- `タンク` => 各陣に３つずつあり所持してる`エナジー`を注入するためのもので`タンク`のうちの２つは一定量の`エナジー`を注入すると壊れる  
- `メカ` => 各陣にランダムで出現してゲーム勝利の手助けになるアイテム  
- `ガジェット` => キャラクターが3つまで所持できるものでステータスを変化させることができる  
- `ロボ` => フィールドに出現する野良モンスターで倒すと`エナジー`と経験値を獲得できる  
<br>  
<br>  


### **メカの種類[^3]**
- 高速でスライド移動できるシューズ  
- 相手の攻撃を無効化できるシールド  
- 建物を自由自在に移動できるシューズ  
- 相手の移動速度を減速できるなにか  
- 味方全体のエナジーを回復できるキット  
- フィールドに設置できる地雷  
- 物体を透過して敵の位置を特定できるゴーグル  
- 敵を行動不能にできるパルス  
- 透明になれるスーツ  
- 任意の場所にワープできる装置  
<br>  
<br>  


### **ゲーム性**  <br>  

#### **ライフとエナジー**  
- ライフが0になるとフィールドから姿が消え数秒後に自陣のスタート地点で復活[^1]  
- 武器を使うためには`エナジー`を消費する必要がある[^1]  
- `エナジー`が切れると武器を使えない[^1]  
- 味方の`エナジー`量は確認できるが敵の`エナジー`は確認できない  
- フィールドに出現する`ロボ`を倒すと`エナジー`を獲得できる[^2]  
- スタート地点に戻るとライフと`エナジー`を回復できる[^1]  
- フィールドの特定の場所に行くとスタート地点に戻れる[^3]  
- 敵を倒すと相手の残ってたエナジーを全部取得できる[^1]  
<br>  

#### **ロボ**  
- ロボットが出現したらミニマップに位置が表示される[^3]  
- モブロボットはランダムな場所に一定時間おきに出現[^3]  
- 一定時間おきにフィールドにボスロボットが出現し最後の一撃を与えたチーム全体にエナジーを与える[^3]  
<br>  


#### **勝敗の決め方**  
- 敵陣の一番奥に1個その少し手前に2個の`タンク`がある[^1]  
- 手前の2個の`タンク`は一定量の`エナジー`を注入すると壊れる[^2]  
- 一番奥のタンクには無制限に`エナジー`を注入できる[^2]  
- `エナジー`をより多く注入したチームの勝ち[^1]  
- 自分たちが注入したエナジーの総量はゲーム終了までわからない[^2]  
<br>  


#### **位置情報**  
- バトル中の画面では左上にマップが小さく表示される[^3]  
- 敵の位置は基本わからない[^2]  
- 味方が敵に一定距離近づいたらミニマップに敵の位置が表示され味方と位置情報を共有できる[^3]  
- 味方の位置は常にミニマップに表示される[^3]  
- ロボットを敵が倒したらその場所に近づくまでまだ存在するかわからない[^3]  
- タンクが壊れる前に敵陣側のフィールドに入ると敵に位置がバレる[^3]  
<br>  

#### **レベル**  
- ロボットや敵を倒すと経験値が入る[^3]  
- 一定量の経験値が溜まるとレベルが上がる[^3]  
- レベルには上限がある[^3]  
- 一定量のレベルに達すると必殺技を覚える[^4]  
<br>  

#### **その他**  
- メカはフィールドの中心寄りの場所にランダムに一定時間で出現[^4]  
- ゲーム時間は8分[^1]  
<br>  
<br>

### **操作性** 
- 遠距離系武器は武器ボタンを押すとエイミングになり移動スティックでエイムし武器ボタンを離すと発射される  
- 近距離系武器は武器ボタンを押すと武器を振り下ろす  
- 近距離系攻撃は連続で発動するとモーションが変わる  
- 武器を複数回使うと強化攻撃が出る  
- 移動スティック上で前に下で後ろに左右で左右に動く  
- 何もないとこをスライドすると視界が動く  
- 必殺技ボタンは覚えるレベルに達するまでは半透明  
- 必殺技ボタンを一度使うと次に溜まるまでの時間が表示される
